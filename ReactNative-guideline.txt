
*** Các bước cài đặt môi trường phát triển React Native cho target Android trên máy Windows ***
https://reactnative.dev/docs/environment-setup
1, Cài đặt JDK, sau đó thêm biến môi trường JAVA_HOME với value là Path của thư mục JDK vừa cài
2, Cài Node thông qua nvm-windows:  https://github.com/coreybutler/nvm-windows
2.1 Tải bộ cài nvm (Node Version Manager) về và cài đặt;
Chú ý để Path là mặc định "C:\Users\{username}\AppData\Roaming\nvm" để tránh bị lỗi phân quyền.
2.2 Sau đó mở Windows Powershell, cài đặt node.js với command
> nvm install latest
và kiểm tra lại với lệnh > nvm list; > nvm use {version}
3, Tải về và cài đặt Android Studio theo hướng dẫn, chú ý:
- Cài đặt Android SDK và thêm biến môi trường ANDROID_HOME với value là Path của thư mục SDK vừa cài;
- Cài đặt Android SDK Platform (có lấy thêm cả bản Android 10 (Q) như hướng dẫn)
- Thêm SDK\platform-tools vào System Path
4, Thiết lập kết nối đến thiết bị Android: xem https://reactnative.dev/docs/running-on-device
4.1 Enable Debugging over USB
4.2 Plug in your device via USB
4.3 Connecting to the development server -> Method 1: Using adb reverse

5, Tải và cài đặt React Native Command Line Interface
bằng cách chạy một lệnh npx react-native <command> , vd như sau
Mở Windows Powershell:
> nvm list
> nvm use 16.0.0
> npx react-native init QuanLyDocSach


*** Sử dụng ***

Mở Windows Powershell hoặc Terminal với quyền Admin:
> nvm list
> nvm use 16.0.0
> npx react-native init QuanLyDocSach
> cd QuanLyDocSach
> npx react-native start
hoặc chỉ cần lệnh
> npm start

Mở Windows Powershell thứ hai:
> nvm use 16.0.0
> cd QuanLyDocSach
> npx react-native run-android

*** Cài đặt package mới cho TabView ***
https://www.npmjs.com/package/react-native-tab-view
Mở Windows Powershell
> npm install -g yarn
Sau đó mở Terminal và cài đặt các package vào thư mục của app
> nvm use 16.0.0
> cd QuanLyDocSach
> yarn
> yarn add react-native-tab-view
> yarn add react-native-pager-view
rồi sau đó tắt Terminal đi, và bật lại 2 màn hình Terminal dùng các command start r.n và run app như trên


*** Xử lí khi gặp lỗi ***
1, Lỗi "Module AppRegistry is not registered callable module (calling runApplication)"
=> chạy lệnh > npm start -- --reset-cache
